## 11. Градационные алгоритмы обработки изображений

**Цель работы**: 
    Получить практические навыки программной реализации градационных
алгоритмов типовых преобразований изображений.

**Программа работы**
1. Разработать экранный интерфейс программы преобразований изображений,
обеспечивающей:
    * возможность ввода изображений из файлов форматов BMP, PCX, PNG (на выбор) и их
отображения на мониторе
    * ввод значений параметров преобразования
    * реализацию заданных видов преобразований изображения и их вывод на монитор
2. Реализовать заданные преобразования изображения
3. Проанализировать полученные изображения

**Необходимые функции работы программы** 
 
   * Преобразование цветного изображения в полутоновое
   * Преобразование полутонового изображения в бинарное (черно- белое)
   * Преобразование изображения в негатив
   * Логарифмическое преобразование
   * Степенное преобразование
